Test Suite v1

These files test various features of the language.
The list of files will increase and change as features and specs are added.

As of 10/13/21 the list includes:

Basic math testing with all Types excluding String.
    - Addition
    - Subtraction
    - Multiplication
    - Division
    - Modulus

String functionality:
    - String concatenation
    - String comparison
    - Any other math operation resulting in a NaN

Functions:
    - Local variables
    - Function variables
    - Function return values

Conditionals:
    - True statements cause conditionals to run code
    - False statements do not cause conditionals to run code

Interval functionality:
    - Breakouts exiting intervals
    - Breakouts ceasing execution of code
    - Intervals and infinite loops execute properly
